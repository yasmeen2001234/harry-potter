⚙️ Firebase Setup Required

To enable online multiplayer, you need a free Firebase project. Here's how:

Step 1. Go to console.firebase.google.com → Create a project (disable Analytics for speed).

Step 2. Click Realtime Database → Create database → Start in test mode.

Step 3. Click the ⚙️ gear → Project settings → Add app → Web → copy your config.

Step 4. Replace the FIREBASE_CONFIG object at the top of this file's <script> with your values:

const FIREBASE_CONFIG = { apiKey: "AIza...", authDomain: "your-project.firebaseapp.com", databaseURL: "https://your-project-default-rtdb.firebaseio.com", projectId: "your-project", storageBucket: "your-project.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc" };

💡 The Realtime Database free tier (Spark plan) supports 100 simultaneous connections and 1 GB storage — plenty for this game.
